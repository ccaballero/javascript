Funciones de cadena

var s = "hello, world"
s.charAt(0)          // => "h"
s.charAt(s.length-1) // => "d"
s.substring(1,4)     // => "ell"
s.slice(1,4)         // => "ell"
s.slice(-3)          // => "rld"
s.indexOf("l")       // => 2
s.lastIndexOf("l")   // => 10
s.indexOf("l", 3)    // => 3
s.split(", ")        // => ["hello", "world"]
s.replace("h", "H")  // => "Hello, world"
s.toUpperCase()      // => "HELLO, WORLD"

Hoisting

Las variables en JavaScript, tienen ambito de función, esto significa que todas las variables declaradas en una función son visibles a travez del cuerpo de la función. Esto significa que las variables son incluso visibles antes de ser declaradas.

Funciones

Las funciones se declaran con la palabra reservada función.

function f(x,y){
    return x+y
}

Caracteristicas

Una función puede ser asignada a una variable.

var f=function(x){
    console.log(x)
}
f(10)

Caracteristicas

Como una función puede ser asignada a una variable, es posible que una función reciba otras funciones como parte de sus parametros.

function f(x,g){
    return g(x)
}
var a=f(10,function(x){return x*2})
console.log(a) // 20

Caracteristicas

Ademas una función puede definirse de forma anonima, ademas de ejecutarse en la misma sentencia de su definición.

var a=(function(x,y){
    return x+y
}(1,2))

Caracteristicas

Otra caracteristica importante sobre las funciones en JavaScript, es que puede definirse una función dentro de otra función.

function a(){
    var x=10
    function b(){
        return x+1
    }
    return b()
}
console.log(a())

Varargs

Al igual que en el lenguaje de programación Java, una función puede ser invocada con mas parametros de los que han sido definidos, o inclusive con menos.

Todas los parametros que han sido enviados a una función pueden ser accedidos a esta por el arreglo objeto "arguments".

function a(){
    console.log(arguments)
}
a(1,2,3,4,5,6)

Objetos

Un objeto es una valor compuesto, que agrega multiples valores, ya sean primitivas o otros tipos de colecciones.

Un objeto en JavaScript puede ser definido de 3 maneras.

Definición literal

Es una definición de propiedades de tipo clave:valor.

a = {
    x:1,
    y:[1,2,3,4,5],
    z:function(a){return a}
}

New

Otra manera de definir un objeto es instanciando una clase de un tipo de algun Objeto definido en JavaScript.

a = new Object()
b = new String('asdf')

Objetos como arreglos asociativos

Las variables de un objeto pueden ser accedidas como si fuesen arreglos asociativos:

a={x:1,y:2}
console.log(a.x)
console.log(a['x'])

Clases

Una clase en JavaScript es una colección de objetos que heredan propiedades de un mismo prototipo.

function A(){
    this.a=10
    this.b=11
}
a=new A()

Prototipos

Es un objeto asociado al objeto que se esta definiendo, y del cual va a heredar propiedades.

Es establecido cuando el objeto es creado.

Usos

Se usa para agregar funciones a una definición incluso despues de haber declarado alguna instancia.

a=new String('asdf')
String.prototype.saludar=function(){
    return 'hola'
}
console.log(a.saludar())

Usos

Tambien pueden servir para realizar herencia entre objetos.

function A(){
    this.a=100
    this.x=function(){
        return this.a
}}
function B(){this.b=100}
B.prototype=new A()
b=new B()
console.log(b.x())

Object.create

Construye un nuevo objeto a partir de un prototipo establecido.

function A(){
    this.f=function(){return 'asdf'}
}
a=new A()
b=Object.create(a)
console.log(b.f())

Object.keys

Retorna un arreglo de nombres de las propiedades enumerables de un objeto.

function A(){
    this.x=1
    this.y=2
}
console.log(Object.keys(new A()))

Object

La clase Object, posee varios metodos:

  • toString()

    Retorna la representación en cadena de un objeto.

  • valueOf()

    Retorna la forma primitiva de un objeto.

JSON

JSON, acrónimo de JavaScript Object Notation, es un formato ligero para el intercambio de datos. JSON es un subconjunto de la notación literal de objetos de JavaScript que no requiere el uso de XML.

Ejemplo

{"menu": {
    "id": "file",
    "value": "File",
    "popup": {
        "menuitem": [
            {"value": "New", "onclick": "CreateNewDoc()"},
            {"value": "Open", "onclick": "OpenDoc()"},
            {"value": "Close", "onclick": "CloseDoc()"}
        ]
    }
    }}

Transformación

Existen dos métodos para la transformación entre JSON y Objetos de JavaScript

JSON.stringify OBJETO -> JSON

Retorna una cadena en formato JSON correspondiente a un valor especificado.

JSON.parse JSON -> OBJETO

Interpreta una cadena en formato JSON a su objeto correspondiente.

DOM

DOM (Document Object Model), es una interfaz de programación para documentos en HTML y XML. Esta provee una representación estructurada del documento (en forma de una estrucura de arbol), y define una manera para que la estructura pueda ser accedida, ademas de poder modificar su estilo o contenido.

El DOM permite un acceso a la estructura de una página HTML mediante el mapeo de los elementos de esta página en un árbol de nodos. Cada elemento se convierte en un nodo y cada porción de texto en un nodo de texto.
